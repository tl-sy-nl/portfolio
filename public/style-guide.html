Color System — Optimization

問題診斷：目前有 11 個色彩變數，其中 ink 三階全是近黑色（13.9:1 → 14.2:1 → 17.5:1）看不出層級；gold 三階差距太小（4.67:1 → 5.77:1 → 6.91:1）視覺上是同一個顏色。

提案核心：砍到 7 個。讓每個顏色都有不可替代的角色。

1. 問題：現有系統為什麼不好看

現狀 — 11 個變數

Ink 三階

三個都是「黑色」——13.9:1 / 14.2:1 / 17.5:1
眼睛分不出來，層級消失

Gold 三階

三個都是「咖啡金」——4.67:1 / 5.77:1 / 6.91:1
功能不同但看起來一樣

問題

· ink 層級倒反（primary 最淺、tertiary 最深）
· 沒有呼吸空間——全是深色擠在一起
· 蒂芬尼藍被限制在深底上，浪費了最漂亮的顏色
· 11 個變數但視覺上只有 3 種感受（黑、金、藍）

提案 — 7 個變數

Ink 兩階（有真正的層級差）

#211E55 (15.1:1) → #5C5980 (6.0:1)
一看就知道誰是主要、誰是次要

Gold 兩階（文字 + 裝飾）

#8A6838 做文字 (4.67:1) / #C4956A 只做裝飾底色
角色清楚，不會混淆

優勢

· 每個顏色有不可替代的功能
· Hover 用 ink（深色取代金色）——更乾淨
· 蒂芬尼藍作為全站「寶石色」有更多出場空間
· 變數少、維護容易

4. 顏色角色對照表（最終版）

角色 變數 色值 對比度 使用範圍
主要文字 --ink #211E55 15.1:1 body copy、h1、pro-item title、case-meta value
次要文字 --ink-muted #5C5980 6.0:1 描述、subtitle、公司名、caption、period、sub-label、arrow
強調 / 互動 --accent #8A6838 4.67:1 連結、eyebrow、職稱、NDA、figcaption、meta label、Contact em
裝飾底色 --accent-glow #C4956A — tag/pill rgba 底色、callout 上邊線、dot active（永不做文字）

6. 需要改動的地方

1

合併 --ink-2 和 --ink-3 為 --ink-muted (#5C5980)

刪掉兩個近黑色，用一個有真正層級差的靛藍灰取代。

2

合併 --accent-mid 和 --accent-warm 為 --accent (#8A6838)

Hover 用 --ink（深色）而非另一個金色。

3

Hero 四行詩文字改用蒂芬尼藍 rgba

從 rgba(220,235,248,0.85) → rgba(126,200,190,0.7)，讓 hero 和 About 有色彩呼應。

4

刪除 --accent-light (#D4A87E)

深色底上改用 --tiffany，不再需要金色。

5

Case impact bar 標籤改用蒂芬尼藍

Impact bar 背景是 slate 調的暗底，蒂芬尼藍在上面 OK（深底限定規則）。
